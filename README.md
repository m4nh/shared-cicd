# shared-cicd

A collection of reusable GitHub Actions and workflows for CI/CD automation, with a focus on Python projects, Docker builds, and semantic versioning.

## Overview

This repository provides:

- **Reusable GitHub Actions** for common build, test, and release tasks
- **Reusable Workflows** that orchestrate complex CI/CD pipelines

Perfect for teams that need consistent, maintainable CI/CD across multiple projects.
